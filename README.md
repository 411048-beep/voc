# 📚 Vocab Master - 背單字應用

一個純前端的英文背單字應用，無需任何框架，使用 HTML、CSS 和 JavaScript 構建。

## 🎯 功能特性

### 1. **卡片翻頁學習頁面**
- 中央卡片設計，點擊可產生翻頁效果
- 正面顯示英文單字
- 背面顯示：
  - 中文翻譯
  - 詞性（noun, verb, adjective 等）
  - 例句
  - 字根分析/詞源

### 2. **單字管理頁面**
- 新增英文單字及其資訊
- 支持自動填入功能
- 查看所有已新增的單字
- 刪除不需要的單字

### 3. **自動填入功能** ⭐
- 在管理頁面輸入英文單字後，點擊「自動填入」按鈕
- 自動調用免費的 Dictionary API 獲取：
  - 中文翻譯（英文定義）
  - 詞性
  - 例句
  - 字根/詞源資訊
- 無需手動查詢和輸入

### 4. **其他功能**
- 單字導航：上一個/下一個
- 進度條顯示當前進度
- 本地存儲：使用 localStorage 保存數據（不需要後端）
- 響應式設計：支持桌面和移動設備
- 單字計數和統計

## 📋 使用方法

### 安裝和運行
1. 下載或克隆此項目
2. 在瀏覽器中直接打開 `index.html` 文件
3. 無需任何服務器或環境配置

### 學習流程
1. **首先進入管理頁面**
   - 輸入英文單字
   - 點擊「自動填入」自動獲取信息
   - 或手動輸入翻譯、詞性、例句、字根
   - 點擊「新增單字」保存

2. **進入首頁開始學習**
   - 查看卡片正面的英文單字
   - 點擊卡片翻頁查看背面信息
   - 使用左右按鈕導航到其他單字
   - 觀看進度條追蹤學習進度

  - Flexbox 佈局

  - 非同步/等待（async/await）

## 📡 API 整合

本應用使用免費的 **Dictionary API**：
- **基礎 URL**: `https://api.dictionaryapi.dev/api/v2/entries/en/`
- **無需認證**：完全免費，無配額限制
- **返回信息**：定義、詞性、例句、詞源等

**示例**：
```
https://api.dictionaryapi.dev/api/v2/entries/en/hello
```

## 💾 數據存儲

- 所有單字都保存在瀏覽器的 `localStorage` 中
- 數據格式：JSON 序列化
- 數據持久化：刷新頁面後數據仍然存在
- 跨會話保存：關閉瀏覽器後數據仍然保存

## 🚀 快速開始

1. 打開 `index.html` 在瀏覽器中
2. 點擊「管理」進入管理頁面
3. 輸入英文單字（例如：`hello`）
4. 點擊「自動填入」按鈕自動填充信息
5. 點擊「新增單字」保存
6. 點擊「首頁」開始學習卡片

## 💡 使用技巧

1. **快速查詞**：自動填入功能支持任何英文單字
2. **自定義編輯**：自動填入後仍可手動修改任何字段
3. **批量添加**：重複步驟 3-5 來快速添加多個單字
4. **數據備份**：打開瀏覽器控制台，運行：
   ```javascript
   JSON.stringify(JSON.parse(localStorage.getItem('vocabWords')), null, 2)
   ```
   複製結果保存為備份

## 🔒 數據隱私

- 不上傳到任何服務器
